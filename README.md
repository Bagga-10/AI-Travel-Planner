✈️ AI Travel Planner

An AI-powered Personalized Travel Planner that generates travel itineraries and provides real-time travel assistance using AI. Built with React.js, Express.js, Firebase, and Machine Learning.

🌟 Key Features

✅ AI-Powered Travel Itineraries – Get personalized trip plans based on your preferences.

✅ AI Chatbot for Travel Queries – Ask questions and get instant travel assistance.

✅ User-Friendly UI – A sleek, modern interface with a seamless user experience.

✅ Seamless Backend Integration – Built with Express.js and Google Firebase for efficiency.

🛠️ Tech Stack

Frontend: React.js + Vite

Backend: Node.js + Express.js

Database: Google Firebase

Machine Learning: AI-based trip recommendations

APIs Used:

🌍 OpenWeather API (Weather updates)

📍 Google Places API (Location details)

🤖 Google Gemini API (AI chatbot)

🔥 Google Firebase API (User data & storage)

🚀 Getting Started

1️⃣ Clone the Repository

git clone https://github.com/Bagga-10/AI-Travel-Planner.git

cd ai-travel-planner

2️⃣ Setup the Backend (Node.js)

cd Backend

npm install

node server.js

3️⃣ Setup the Frontend (React.js)

cd Frontend

npm install

npm run dev

4️⃣ Configure Environment Variables

Create a .env.local file in the Frontend directory:

VITE_GOOGLE_PLACE_API_KEY=your_api_key

VITE_GOOGLE_GEMINI_AI_API_KEY=your_api_key

VITE_GOOGLE_AUTH_CLIENT_ID=your_auth_id

VITE_FIREBASE_API_KEY=your_api_key

VITE_BACKEND_URL=your_backend_url

Create a .env file in the Backend directory:

GEMINI_API_KEY=your_api_key

FRONTEND_URL=your_frontend_url

PORT=your_port


🔍 How to Preview

1️⃣ Start the backend server

node server.js

2️⃣ Start the frontend

npm run dev

3️⃣ Open http://localhost:5174/ in your browser 🚀


🎯 Now, interact with the AI chatbot and get personalized itineraries! 🌍
